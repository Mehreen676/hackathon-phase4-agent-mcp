## Phase
Phase II: Full-Stack Web Application

## Goal
Transform the Todo console app into a professional, multi-user web application using
Spec-Driven Development.

## Core Principles
- Spec-first, code-later approach
- No manual coding without finalized specifications
- Clean architecture with clear separation of frontend and backend
- Professional, responsive, and modern UI

## Tech Stack
### Frontend
- Next.js (App Router)
- TypeScript
- Tailwind CSS

### Backend
- Python FastAPI
- SQLModel ORM

### Database
- Neon Serverless PostgreSQL

### Authentication
- Better Auth (JWT-based)

## Functional Scope (Phase II)
- Create, read, update, delete tasks
- Mark tasks as complete/incomplete
- Each user can only access their own tasks
- RESTful API with JWT protection
- Responsive and professional web UI

## Out of Scope (Later Phases)
- AI chatbot
- Kubernetes
- Kafka / Dapr
